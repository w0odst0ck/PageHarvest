# 1688 智能照明商品数据采集 → 分析 工作流

## 一、数据采集 → 处理 → 分析 完整链路

```
 你 (浏览器操作)              脚本 (自动处理)
 ──────────────────          ──────────────────
                             
 ① 打开1688搜索关键词
         │
         ▼
 ③ 油猴自动翻页34页                 ← 01_fetch.user.js
    + 滚动加载全部商品
    + 下载HTML到浏览器下载目录
         │
         ▼
 ④ 把HTML移到data/目录              ← 手动操作
         │
         ▼
                                ⑤ 02_parse.py
                                  解析HTML → CSV (17字段)
                                        │
                                        ▼
                                ⑥ 03_clean.py
                                  数据清洗 → 去重+修正 (342条)
                                        │
                                        ▼
                                ⑦ 04_analyze.py
                                  分析出图 → 报告+图表
                                        │
                                        ▼
                                  data/analysis_report.txt
                                  data/analysis_chart.png
```

## 二、你的操作 vs 脚本自动处理

### 你手动做的事
| 步骤 | 操作 | 频率 |
|------|------|------|
| ① 油猴安装 | 把 `01_fetch.user.js` 拖进 Tampermonkey | 一次 |
| ② 搜关键词 | Chrome打开1688输入关键词 | 每品类一次 |
| ③ 确认下载 | 弹窗点确认，让油猴自动翻页 | 每品类一次 |
| ④ 移HTML | 从下载目录移到 `data/关键词/` | 每品类一次 |

### 脚本自动做的事
| 步骤 | 脚本 | 作用 |
|------|------|------|
| ⑤ 解析 | `02_parse.py` | HTML→CSV，17个字段全自动 |
| ⑥ 清洗 | `03_clean.py` | 去重、修价格、补品牌、标准化 |
| ⑦ 分析 | `04_analyze.py` | 价格分布、供应商排名、品类分析、图表 |

## 七、注意事项

1. **油猴一次只跑一个关键词**：跑完一个再搜下一个
2. **URL编码**：1688用GBK编码，油猴脚本已处理
3. **下载被阻止**：Chrome提示时点"允许"即可
4. **数据去重**：同一商品跨页面出现时会被合并
5. **品牌缺失**：搜索页无品牌字段，从标题中匹配已知品牌
