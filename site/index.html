使用流程

从采集到报告，四个简单步骤

安装油猴脚本

安装浏览器脚本，自动翻页采集搜索页 HTML，或手动 Ctrl+S 保存详情页

打包 ZIP

将保存的 HTML 文件打包成一个 ZIP 压缩包，文件命名不限

上传解析

通过 exe 应用或 CLI 上传 ZIP，工具自动识别平台并提取商品数据

下载报告

获取结构化 CSV/Excel 报告，直接用于选品分析和决策

产品策略

离线分析 = 产品，在线采集 = 开源

产品

离线解析（GitHub Pages）

上传 ZIP 即用，纯静态页面 + GitHub Actions 后台解析。零部署，零费用，数据不上传第三方。

本地 exe

本地客户端

PyInstaller 打包的单文件 exe，双击运行。无需网络，完全离线，隐私敏感用户首选。

开源

在线采集（自部署）

在线采集代码全部公开，有技术能力的用户可自行部署。适用于需要实时数据采集的场景。
