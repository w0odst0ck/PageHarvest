# 1688 智能照明商品数据采集 & 分析工具

> 一键采集 1688 智能照明品类商品数据，自动解析、清洗、分析，输出可视化报告。

---

## 🚀 功能

- 📥 **数据采集** — 油猴脚本自动翻页 + 滚动加载，采集 1688 搜索结果页全部商品
- 📄 **HTML → CSV 解析** — 自动提取 17 个字段，从原始页面到结构化表格
- 🧹 **数据清洗** — 去重、价格标准化、品牌识别、标题修正
- 📊 **数据分析** — 价格分布、供应商排名、品类分析、经营时长、回头率，输出文本报告 + 可视化图表
- ⚡ **一键运行** — 多步骤管线可全自动执行

## 📂 项目结构

```
1688-scrapy/
├── pipeline/                  # 全流程管道
│   ├── 01_fetch.user.js       # 油猴脚本 — 翻页 + 滚动 + 下载 HTML
│   ├── 02_parse.py            # HTML → CSV 解析（17 字段）
│   ├── 03_clean.py            # 数据清洗（去重 + 价格 + 品牌）
│   └── 04_analyze.py          # 数据分析 + 图表生成
│
├── data/                      # 数据文件
│   ├── 户外灯具/               # 原始 HTML（一个品类一个文件夹）
│   ├── all_products.csv       # 解析后的原始 CSV
│   ├── cleaned_products.csv   # 清洗后可直接用 Excel 打开的数据
│   ├── analysis_report.txt    # 分析报告
│   └── analysis_chart.png     # 可视化图表
│
└── requirements.txt           # Python 依赖
```

### 快速开始

```powershell
# 1. 安装依赖
pip install -r requirements.txt

# 2. 采集数据（浏览器操作）
#    - 安装 01_fetch.user.js 到 Tampermonkey
#    - 打开 1688 搜索关键词
#    - 手动将下载的 HTML 移到 data/品类名/

# 3. 一键分析
python pipeline\02_parse.py    # 解析 HTML → CSV
python pipeline\03_clean.py    # 数据清洗
python pipeline\04_analyze.py  # 生成报告 + 图表
```

## 📊 分析输出

- **价格分布** — 各价格区间的商品数量统计
- **供应商排名** — 按商品数排名的供应商列表
- **品类分布** — 品类间商品数量对比
- **店铺经营分析** — 经营时长分布、回头率统计
- **可视化图表** — 一张图表汇总关键数据

## 🛠 技术栈

- **油猴脚本 (Tampermonkey)** — 浏览器端自动化翻页采集
- **Python 3** — 数据解析、清洗、分析
- **BeautifulSoup4 / lxml** — HTML 解析
- **matplotlib** — 可视化图表
- **pandas / csv** — 数据处理

## ⚠️ 注意事项

1. 油猴一次只跑一个关键词，跑完再搜下一个
2. 1688 使用 GBK 编码，油猴脚本已内置处理
3. 浏览器下载弹窗时需手动点击确认
4. 同一商品跨页面出现时会被自动去重合并
5. 品牌数据目前从标题中匹配已知品牌（详情页品牌字段待后续采集）
