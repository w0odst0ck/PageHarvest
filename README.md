# 1688 智能照明商品数据采集 & 分析工具

> 一键采集 1688 智能照明品类商品数据，自动解析、清洗、分析，输出可视化报告。

---

## 🚀 功能

- 📥 **数据采集** — 油猴脚本自动翻页 + 滚动加载，采集 1688 搜索结果页全部商品
- 📄 **HTML → CSV 解析** — 自动提取 17 个字段，从原始页面到结构化表格
- 🧹 **数据清洗** — 去重、价格标准化、品牌识别、标题修正
- 📊 **数据分析** — 价格分布、供应商排名、品类分析、经营时长、回头率，输出文本报告 + 可视化图表
- ⚡ **一键运行** — 多步骤管线可全自动执行

## 📂 项目结构

```
1688-scrapy/
├── pipeline/                  # 全流程管道
│   ├── 01_fetch.user.js       # 油猴脚本 — 翻页 + 滚动 + 下载 HTML
│   ├── 02_parse.py            # HTML → CSV 解析（17 字段）
│   ├── 03_clean.py            # 数据清洗（去重 + 价格 + 品牌）
│   └── 04_analyze.py          # 数据分析 + 图表生成
│
├── data/                      # 数据文件
│   ├── 户外灯具/               # 原始 HTML（一个品类一个文件夹）
│   ├── all_products.csv       # 解析后的原始 CSV
│   ├── cleaned_products.csv   # 清洗后可直接用 Excel 打开的数据
│   ├── analysis_report.txt    # 分析报告
│   └── analysis_chart.png     # 可视化图表
│
└── requirements.txt           # Python 依赖
```

## 🛠 技术栈

## ⚠️ 注意事项

1. 油猴一次只跑一个关键词，跑完再搜下一个
2. 1688 使用 GBK 编码，油猴脚本已内置处理
3. 浏览器下载弹窗时需手动点击确认
4. 同一商品跨页面出现时会被自动去重合并
5. 品牌数据目前从标题中匹配已知品牌（详情页品牌字段待后续采集）

## 各步骤说明

### 01_fetch — 数据采集
- 油猴脚本，在1688搜索页自动翻34页
- 每页滚动加载全部商品，保存HTML到下载目录
- 手动把HTML移到 `data/关键词/` 文件夹

### 02_parse — HTML解析
- 解析 `search-offer-item` 商品卡片
- 提取 `plugin-offer-search-card` 插件数据
- 输出17个字段到 CSV

### 03_clean — 数据清洗
- 价格格式标准化
- 标题去冗余后缀
- 基于关键词的品牌识别
- 重复商品去重 (标题+供应商)

### 04_analyze — 数据分析
- 价格分布分析 (区间+统计)
- 供应商排名
- 品类分布
- 店铺经营时长
- 回头率分析
- 生成 `analysis_report.txt` 和 `analysis_chart.png`
